# **App Name**: Dropstream

## Core Features:

- Product Catalog: Display products with details: name, description, images, categories, and prices.
- Supplier Directory: Maintain a directory of suppliers with contact information, shipping policies, and return conditions.
- Order Processing: Record the reception and forwarding of orders, then track order status updates in real time.
- Restock Alerting Tool: Use inventory data to determine when the restock level of a product offered by a given vendor has reached a critical level.
- Return Management: Process to register and manage return requests from customers. Provide clear policies.
- User Authentication: Secure authentication for different user roles with specific permissions.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4) to inspire confidence and reliability, suitable for a professional business application.
- Background color: Light gray (#F5F5F5), a very low saturation of the primary color, to ensure legibility and a clean interface.
- Accent color: Yellow (#FFC107) to highlight important actions and notifications.
- Headline font: 'Poppins', a sans-serif font, providing a modern and clean look.
- Body font: 'Inter', a sans-serif font, providing a neutral and readable feel for text-heavy content.
- Use flat, line-based icons to maintain a modern and clean aesthetic.
- Emphasize a clear, logical layout with whitespace to create an efficient and professional UI.
